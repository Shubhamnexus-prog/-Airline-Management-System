# ✈️ Airline Management System

## 📌 Project Description
The Airline Management System is a desktop-based application developed using Java. 
This project helps airlines manage flight schedules, passenger details, and ticket bookings efficiently.

The system provides an easy-to-use interface for airline staff to manage airline operations.

---

## 🛠 Technologies Used
- Java
- Swing GUI
- JDBC
- MySQL Database

---

## ✨ Features
✔ Add Flight Details  
✔ View Flight Information  
✔ Book Flight Tickets  
✔ Passenger Details Management  
✔ Cancel Ticket  
✔ Flight Schedule Management  

---


## 🚀 How to Run the Project

1️⃣ Install Java JDK  
2️⃣ Install MySQL Database  
3️⃣ Import the project database  
4️⃣ Open the project in NetBeans  
5️⃣ Run the main class
